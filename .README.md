o	Titre du projet: Détection automatisée des attaques de phishing

o	Description:

Ce projet porte sur l'étude et l'analyse des techniques de phishing utilisées pour tromper les utilisateurs en ligne. L'objectif principal est de développer des mécanismes de détection automatique afin d'identifier les menaces avec précision.

À travers ce dépôt, nous organisons les données, les analyses Excel et les documents de recherche nécessaires. Ce travail contribue à renforcer la sécurité informatique et à protéger les données sensibles contre les cyberattaques.

Enfin, il permet de sensibiliser les utilisateurs aux risques numériques actuels.
